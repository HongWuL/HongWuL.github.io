PPTPP:A novel therapeutic peptide prediction method using physicochemical property encoding and adaptive feature representation learning

Abstract

PPTPP是一个基于理化属性的治疗性多肽预测器。PPTPP具有的功能：多肽分类以及识别多肽的informative理化性质。

Introduction

研究本课题的意义: 多肽在生理上的广泛用途是对其特性和分类进行深入研究的动力。过去几十年有超过7000种多肽被识别。多肽很短，通常由<50个氨基酸组成，但是是有效的有前途的治疗和诊断工具。基于Wetlab(传统生物学)的实验，有复杂性高和成本高的特点。幸运的是，随着实验数据越来越多，基于机器学习的方法已变得可行…

Methods

Datasets

AAindex

AAindex用于数字化表示544种理化性质，除掉null值还剩531种。

PP based descriptor

PP(理化性质)描述符，输入序列用AAindex表示，然后分别获得它的一阶邻接差分序列和二阶邻接差分序列，然后这三种序列分别计算均值(variance)、方差(variance)、偏度(skewness)、峰度(kurtosis)四个统计量。最后，一个序列由4维向量表示，一个多肽由12维向量表示。

Random Forest

使用WEKA 3.8库。

Adaptive feature learning and assessing

特征选择上，使用 MRMD 2.0 (Maximum-Relevance-maximum-distance) ，其他的类似技术还有ANOVA, MRMD, mRMR。序列排序后使用 SFS(sequential forward search)根据AUC优化。

他是构建了531个meta predictors吗？每个多肽序列得到531个12维特征向量，然后构建531个RF分类器，每个分类器得到3个表示(分类结果，概率值以及两者的融合)。（所以最后是一共构建了531个分类器还是8×531个分类器？

Performance measurement

有点多肽数据集样本数量不平衡，所以AUC是最好的选择。另外，其他4个指标也作为参考:sensitivity (SE), specificity (SP), accuracy (ACC) and Mathew’s correlation coefficient (MCC) 。

Results

Overall performances of meta predictors

测试531个分类器在8个不同多肽数据集上的AUC，即对每个多肽数据集，统计531个分类器得到的AUC的值，然后画出auc-frequency直方图。

其中ABP,ACP,AVP,QSP结果比较好，作者还分析了出现上述结果的可能原因，要么它们的冗余阈值比较大，要么正负样本之间的序列长度差异比较大。因此，很明显，除了肽的内在特性外，数据集是影响本研究构建的元预测器性能的另一个关键因素。

Analysis of adaptive feature learning

使用AUC度量测试adaptive feature learning方法对8种多肽的鉴别能力。每个图由4条roc曲线，分别表示 the best performing single property,the optimal class feature, optimal probabilistic feature and the fusion feature.

结果表明，较表现最好的meta predictor而言，所有的3个feature pools都提高了AUC，而且fusion feature相比概率特征和类别特征AUC提升的更多(PBP除外)。另外还提到作者的结果和ACPred-Fuse一致，后者也用了相同的通过结合概率信息和类别信息构建Fusion feature的策略。然后分析了一下，为什么AUC提升效果如此显著，可能是因为作者提出了相对独立的多肽特征编码方案(指理化性质编码方案)，也可能是 MRMD2.0更Determinant .

Comparison of the proposed predictor and existing predictors

比较对象:PEPPred(目前唯一的多肽综合分类器)，Specie-specific(每个单多肽分类器中最好的方法) ，在十折交叉验证和独立测试集上与本作的三种编码方案分别做比较。

Identification of informative physicochemical features

这是本方法的另一个贡献，即可以顺便判断哪些理化性质特征是informative的(特征选择)。因为特征可以通过MRMD2.0排序.
